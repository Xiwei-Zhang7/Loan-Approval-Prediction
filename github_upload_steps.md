## Upload Method

1. Create a new GitHub repository.
2. Upload all folders and files from this project folder.
3. Make sure the root of the repository contains:

```text
README.md
.gitignore
data/
notebooks/
results/
figures/
report/
source/
```

4. After uploading, open `README.md` on GitHub and check that the figures display correctly.
